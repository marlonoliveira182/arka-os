# Agent YAML Rules

- Every agent must have complete 4-framework behavioral DNA: DISC, Enneagram, Big Five (OCEAN 0-100), MBTI
- Agent type must match a valid type from the core agent schema
- Department field must match the parent directory name
- Tier must be 0 (C-Suite), 1 (Lead), 2 (Specialist), or 3 (Support)
- No duplicate agent names across departments
