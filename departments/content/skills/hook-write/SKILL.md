# Hook Writing — `/content hook <topic>`

> **Agent:** Filipe (Viral Engineer) | **Rule:** Viewer decides in 3 seconds. Hook or die.

## 7 Hook Types

| # | Type | Template | Best For |
|---|------|----------|----------|
| 1 | **Controversy** | "Everyone is wrong about [topic]" | High engagement, comments |
| 2 | **Curiosity Gap** | "I discovered something that changes [outcome]" | Click-through |
| 3 | **Pain Point** | "If you struggle with [problem], watch this" | Problem-aware audience |
| 4 | **Result** | "This got us from [before] to [after] in [time]" | Proof-driven |
| 5 | **Counter-intuitive** | "Stop doing [common advice]. Here's what works" | Authority building |
| 6 | **Story** | "Last [day] I almost [dramatic event]" | Emotional connection |
| 7 | **Direct Value** | "[Number] [things] that will [benefit] in [timeframe]" | Practical, save-worthy |

## Process

1. **Generate 10+ hooks** using all 7 types for the same topic
2. **Score each** on: curiosity (0-3), emotion (0-3), specificity (0-3)
3. **Select top 5** for A/B testing
4. **Format per platform:**
   - YouTube: Title + Thumbnail (hook is visual + text)
   - TikTok/Reels: First spoken words + text overlay
   - LinkedIn: First line before "see more"
   - Twitter/X: First tweet in thread

## Rules

- Under 10 words for video hooks
- Include a number or timeframe when possible
- Never start with "In this video" or "Hey guys"
- The hook IS the content strategy — nail this, rest follows

## Output → 5 hook variants with type, score, and platform formatting
