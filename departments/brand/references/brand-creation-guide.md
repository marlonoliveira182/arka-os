# GUIA COMPLETO DE CRIACAO DE MARCA
## Do Zero ao Nivel das Melhores Marcas do Mundo

> Baseado nas metodologias de **Pentagram**, **Wolff Olins**, **Landor & Fitch**, **Collins**, **DesignStudio**, **Interbrand** e nas praticas internas de marcas como Apple, Nike, Patagonia, Airbnb, Oatly e Liquid Death.

---

## As 8 Fases do Branding Profissional

```
FASE 1 — Pesquisa & Diagnostico     (2-4 semanas)
FASE 2 — Estrategia de Marca        (a fundacao)
FASE 3 — Arquitetura de Marca       (multi-produto)
FASE 4 — Identidade Verbal          (nome, voz, mensagens)
FASE 5 — Identidade Visual          (logo, cores, tipo)
FASE 6 — Aplicacoes & Touchpoints   (digital, print, experiencia)
FASE 7 — Manual de Marca            (documentacao)
FASE 8 — Lancamento & Gestao        (go-to-market)
```

---

## Os 3 Niveis de uma Marca

```
+-------------------------------------+
|         IDENTIDADE VISUAL           |  <- O que as pessoas veem
|    (logotipo, cores, tipografia)    |
+-----------+-------------------------+
|         IDENTIDADE VERBAL           |  <- O que as pessoas ouvem
|      (nome, voz, mensagens)         |
+-----------+-------------------------+
|         ESTRATEGIA DE MARCA         |  <- O que as pessoas sentem
|  (proposito, valores, posiciona.)   |  <- A FUNDACAO (mais importante)
+-------------------------------------+
```

A maioria das pessoas comeca pelo topo. As melhores marcas do mundo comecam pela base.

## FASE 1 — Pesquisa & Diagnostico

### 1.1 Auditoria Interna

**Sobre o negocio:**
- Que problema especifico resolves?
- Para quem resolves esse problema?
- Como ganhas dinheiro?
- O que te diferencia dos concorrentes?
- Qual e a tua vantagem competitiva sustentavel?
- Quais sao os teus pontos fracos honestos?

**Sobre a fundacao:**
- Porque comecaste isto? (A resposta emocional, nao a racional)
- Em que acreditas profundamente sobre o teu setor?
- O que mudaria no mundo se a tua empresa desaparecesse amanha?
- Que comportamentos nao negociaveis tens internamente?

**Exercicio dos 5 Porques (Simon Sinek):**
```
Faco X.
Porque? -> Para conseguir Y.
Porque? -> Porque acredito em Z.
Porque? -> Porque a minha experiencia com W mudou-me.
Porque? -> Porque no fundo quero um mundo onde...
```

### 1.2 Analise da Concorrencia

**3 tipos de concorrentes:**
- Concorrentes diretos — fazem exactamente o que fazes
- Concorrentes indiretos — satisfazem a mesma necessidade de forma diferente
- Concorrentes de atencao — competem pelo mesmo tempo/dinheiro do cliente

**Tabela de Analise Competitiva:**

| Dimensao | Concorrente A | Concorrente B | Tu |
|----------|-------------|-------------|-----|
| Posicionamento | | | |
| Publico-alvo | | | |
| Tom de voz | | | |
| Paleta de cores | | | |
| Preco | | | |
| Pontos fortes | | | |
| Pontos fracos | | | |
| Promessa principal | | | |

**Perceptual Map:** Eixo X e eixo Y com dois atributos opostos relevantes (ex: Tradicional <-> Moderno / Acessivel <-> Premium). Onde esta o espaco vazio? E ai que a oportunidade existe.

### 1.3 Pesquisa de Audiencia

**3 Metodos:**
1. Entrevistas em profundidade (8-12, 45-60min, qualitativas)
2. Surveys (100-500 respostas, quantitativas)
3. Etnografia digital (Reddit, reviews Amazon, comentarios YouTube)

**Jobs-to-be-Done (Clayton Christensen):**
```
"Quando [situacao], quero [motivacao], para que [resultado esperado]."

Exemplo (Airbnb):
"Quando viajo para uma cidade nova, quero sentir-me
em casa e viver como um local, para que a experiencia
seja mais autentica e memoravel."
```

### 1.4 Analise de Tendencias

- PESTLE Analysis — Politico, Economico, Social, Tecnologico, Legal, Ambiental
- Google Trends — Interesse de busca ao longo do tempo
- Industry reports — Statista, IBISWorld, McKinsey

## FASE 2 — Estrategia de Marca

### 2.1 Golden Circle (Simon Sinek)

```
        WHY
    (Proposito/Causa)
       /     \
      /  HOW  \
     /  (Como) \
    /___WHAT____\
      (O que)
```

**Exercicio:** Define o teu WHY em menos de 30 palavras.
```
A [NOME DA MARCA] existe para [PROPOSITO].
Acreditamos que [CRENCA CENTRAL SOBRE O MUNDO].
```

### 2.2 Proposito, Missao, Visao, Valores

**Proposito** — O porque existencial. Eterno. Inspira.
| Marca | Proposito |
|-------|-----------|
| Nike | Trazer inspiracao e inovacao a todos os atletas do mundo |
| Patagonia | Salvar o planeta natal |
| Airbnb | Criar um mundo onde qualquer pessoa pode pertencer a qualquer lugar |
| TED | Disseminar ideias |
| Google | Organizar a informacao mundial e torna-la universalmente acessivel |

**Missao** — O que fazes hoje, para quem, como crias valor.
```
Framework: [EMPRESA] [VERBO DE ACAO] [AUDIENCIA] [RESULTADO/BENEFICIO]
Exemplo: A Tesla [acelera] [a transicao do mundo] [para energia sustentavel].
```

**Visao** — O futuro que queres criar. 5-10 anos. Aspiracional.

**Valores** — Comportamentos nao negociaveis.
- Maximo 5-7
- Cada um deve poder ser usado para tomar uma decisao dificil
- Devem ser especificos, nao genericos
- Devem ser polarizadores

### 2.3 Posicionamento

**Modelo de Geoffrey Moore:**
```
Para [PUBLICO-ALVO]
que [NECESSIDADE OU OPORTUNIDADE]
o/a [NOME DA MARCA] e [CATEGORIA DO PRODUTO]
que [BENEFICIO CHAVE / RAZAO PARA COMPRAR].
Ao contrario de [ALTERNATIVA CONCORRENTE],
o nosso produto [DIFERENCIAL PRINCIPAL].
```

**Brand Ladder:**
```
Nivel 4 -> IDENTIDADE / VALORES     "Sou uma pessoa que..."
Nivel 3 -> BENEFICIO EMOCIONAL       "Faz-me sentir..."
Nivel 2 -> BENEFICIO FUNCIONAL       "Permite-me..."
Nivel 1 -> ATRIBUTO DO PRODUTO       "Tem/E..."
```
As marcas mais poderosas operam nos niveis 3 e 4.

### 2.4 Proposta de Valor (Osterwalder Canvas)

```
+-------------------------+-------------------------+
|   MAPA DE VALOR         |   PERFIL DO CLIENTE      |
|  Gain Creators          |  Customer Gains          |
|  Pain Relievers         |  Customer Pains          |
|  Products & Services    |  Jobs-to-be-Done         |
+-------------------------+-------------------------+
```

### 2.5 Arquetipos de Marca (Carl Jung)

| Arquetipo | Desejo Central | Medo | Marcas |
|-----------|---------------|------|--------|
| O Inocente | Ser feliz | Punicao | Dove, Coca-Cola |
| O Explorador | Liberdade | Conformidade | North Face, Jeep |
| O Sabio | Verdade | Ignorancia | Google, TED |
| O Heroi | Competencia | Fraqueza | Nike, FedEx |
| O Fora da Lei | Revolucao | Impotencia | Harley-Davidson, Virgin |
| O Mago | Transformacao | Estagnacao | Apple, Disney |
| O Cara Normal | Pertenca | Exclusao | IKEA, Target |
| O Amante | Intimidade | Mediocridade | Chanel, Godiva |
| O Bobo | Diversao | Chateza | Ben & Jerry's, Old Spice |
| O Cuidador | Proteger | Ingratidao | Johnson & Johnson, Volvo |
| O Criador | Inovar | Mediocridade | Lego, Adobe |
| O Governante | Controlo | Caos | Rolex, Mercedes |

**Escolha:** Arquetipo primario + secundario (ex: Apple = Mago + Criador).

### 2.6 Territorio de Marca

Define:
- O que a marca fala (temas, assuntos, categorias)
- Onde a marca aparece (contextos, eventos, culturas)
- Com quem a marca se associa (pessoas, outras marcas, causas)

## FASE 3 — Arquitetura de Marca

### 3 Modelos

**House of Brands:** Cada produto e marca independente (P&G, Unilever)
**Branded House:** Uma unica marca-mae (Virgin, FedEx, Google)
**Endorsed Brands:** Sub-marcas com carimbo da mae (Marriott, GM)

## FASE 4 — Identidade Verbal

### 4.1 Nome da Marca

**6 Tipos:**
| Tipo | Descricao | Exemplos |
|------|-----------|---------|
| Descritivo | Descreve o que faz | Facebook, PayPal |
| Evocativo | Evoca ideia/sentimento | Amazon, Apple |
| Inventado | Palavra nova | Kodak, Xerox |
| Acronimo | Iniciais | IBM, BMW |
| Fundador | Nome do fundador | Ford, Disney |
| Metaforico | Metafora | Nike, Jaguar |

**Criterios:** Memoravel, Pronunciavel, Disponivel, Escalavel, Diferente, Significativo

**Processo:** Brief -> 200-500 opcoes -> Filtro -> Lista curta (10-20) -> Due diligence legal -> Teste com audiencia -> Top 3-5

### 4.2 Tom de Voz

**4 Dimensoes (Nielsen Norman Group):**
```
Formal  <--------> Casual
Serio   <--------> Bem-humorado
Respeitoso <-----> Irreverente
Entusiasta <-----> Pragmatico
```

**Exercicios:**
1. Pessoa da marca — Se a marca fosse uma pessoa, quem seria?
2. Dizemos / Nao dizemos — Tabela de exemplos concretos
3. Teste da voz — Remove o logo. Ainda reconheces a marca?

### 4.3 Tagline

**6 Tipos:**
| Tipo | Exemplo |
|------|---------|
| Imperativo | Just Do It (Nike) |
| Descritivo | Connecting People (Nokia) |
| Superlativo | The Ultimate Driving Machine (BMW) |
| Provocatorio | Got Milk? |
| Especifico | 15 minutes could save you 15% (GEICO) |
| Poetico | Finger Lickin' Good (KFC) |

**Criterios:** Memoravel, Inimitavel, Atemporal, Curta (2-5 palavras), Estrategica, Verdadeira.

### 4.4 Messaging Framework

**Piramide de Mensagens:**
```
         PROPOSTA DE VALOR        <- 1 linha
              |
      PILARES DE MENSAGEM         <- 3 pilares
              |
      PROVA (Proof Points)        <- Factos, stats, historias
```

**Mensagens por audiencia:** Clientes, Parceiros, Imprensa, Investidores, Funcionarios.

### 4.5 Storytelling (StoryBrand - Donald Miller)

```
PERSONAGEM (Cliente) -> tem um PROBLEMA -> encontra um GUIA (Marca)
-> que lhe da um PLANO -> que o chama a ACAO -> que resulta em SUCESSO
-> e evita o FRACASSO
```

O cliente e o heroi. A marca e o guia (Yoda, nao Luke).

## FASE 5 — Identidade Visual

### Sistema Completo

```
SISTEMA DE IDENTIDADE VISUAL
|
+-- Logotipo (e variacoes)
+-- Paleta de Cores
+-- Tipografia
+-- Iconografia
+-- Padroes & Texturas
+-- Fotografia (estilo)
+-- Ilustracao (estilo)
+-- Motion (movimento)
+-- Composicao & Layout
```

### 5.1 Logotipo

**7 Tipos:**
| Tipo | Exemplos |
|------|---------|
| Wordmark | Google, Coca-Cola, FedEx |
| Lettermark | IBM, HBO, CNN |
| Symbol/Icon | Apple, Twitter/X, Nike |
| Combination Mark | Amazon, Adidas |
| Emblem | Harley-Davidson, BMW |
| Abstract Mark | Pepsi, Chase |
| Mascot | KFC, Duolingo |

**Principios:** Simplicidade, Memorabilidade, Atemporalidade, Versatilidade, Adequacao

**Teste dos 5 Segundos:** Mostra 5s. Pergunta: O que e? O que faz? Como te faz sentir?

**Variacoes necessarias:**
- Completo horizontal
- Completo vertical/stacked
- So simbolo (favicon, app icon)
- So wordmark
- Versao preto
- Versao branco
- Monocromatico

### 5.2 Paleta de Cores

**Psicologia das Cores:**
| Cor | Associacoes | Marcas |
|-----|-------------|--------|
| Vermelho | Urgencia, energia, paixao | Coca-Cola, Netflix |
| Azul | Confianca, seguranca | Facebook, IBM, PayPal |
| Verde | Natureza, saude | Starbucks, Spotify |
| Amarelo | Otimismo, calor | McDonald's, IKEA |
| Laranja | Amizade, criatividade | Amazon, Fanta |
| Roxo | Luxo, sabedoria | Cadbury, Twitch |
| Rosa | Modernidade, cuidado | T-Mobile, Barbie |
| Preto | Luxo, sofisticacao | Chanel, Nike |
| Branco | Pureza, simplicidade | Apple, Tesla |

**Estrutura:**
- Primaria (1) — 60% da presenca visual
- Secundaria (1-2) — 30%
- Suporte (2-4) — 10%

**Harmonias:** Monocromatica, Complementar, Analoga, Triadica

**Especificacoes para cada cor:** Nome, HEX, RGB, CMYK, Pantone, RAL

### 5.3 Tipografia

**4 Grupos:**
| Grupo | Personalidade | Exemplos |
|-------|---------------|---------|
| Serif | Tradicional, autoridade | Garamond, Georgia |
| Sans-Serif | Moderno, acessivel | Helvetica, Futura |
| Script | Elegante, pessoal | Pacifico |
| Display | Unico, impactante | Bebas Neue |

**Sistema:**
- Primaria (Headlines)
- Secundaria (Body)
- Apoio opcional (citacoes, labels)

**Hierarquia:**
```
H1 -> 48-72px | Primary Font | Bold
H2 -> 32-48px | Primary Font | Medium/Bold
H3 -> 24-32px | Secondary Font | Medium
Body -> 16-18px | Secondary Font | Regular
Caption -> 12-14px | Secondary Font | Regular
Line height: 1.4-1.6x | Max 65-75 chars/line
```

### 5.4 Fotografia & Estilo Visual

Define: Estilo de edicao, Tipo de conteudo, Composicao, Modelos/Pessoas, Ambientes.

**NAO usar:** Fotos de banco com poses artificiais, imagens genericas de handshakes e light bulbs.

### 5.5 Motion & Animacao

Define: Principios de movimento, Transicoes, Logotipo animado, Loops para redes sociais.

## FASE 7 — Manual de Marca

### Estrutura Completa

```
01. INTRODUCAO (carta fundador, como usar, visao geral)
02. ESTRATEGIA (proposito, missao, visao, valores, posicionamento, audiencia, arquetipo)
03. IDENTIDADE VERBAL (nome, tom de voz, tagline, mensagens-chave)
04. IDENTIDADE VISUAL (logotipo, regras, zona exclusao, tamanho minimo, usos proibidos, cores, tipografia, iconografia, fotografia, ilustracao/motion)
05. APLICACOES (digital, print, publicidade, embalagem, sinaletica)
06. RECURSOS (ficheiros download, fontes/licencas, contacto gestor de marca)
```

### Zona de Exclusao (Clear Space)

Espaco minimo obrigatorio em torno do logotipo. Unidade = altura da letra "x" do logotipo.

### Usos Proibidos

Nao distorcer, nao rodar, nao adicionar sombras/efeitos, nao usar em fundos que comprometam legibilidade, nao recriar com fontes diferentes, nao alterar cores, nao adicionar elementos, nao usar versao desatualizada.

## FASE 8 — Lancamento & Gestao

### Difusao de Inovacao (Everett Rogers)

Inovadores (2.5%) -> Early Adopters (13.5%) -> Early Majority (34%) -> Late Majority (34%) -> Laggards (16%)

Conquista os Early Adopters primeiro. Nao tentes comunicar para toda a gente.

### KPIs de Marca

**Awareness:** Brand awareness %, Share of voice, Recall nao-assistido
**Percecao:** NPS, Brand sentiment, Associacoes de marca
**Equity:** Price premium, Brand preference, Customer Lifetime Value

### Evolucao de Marca

| Tipo | O que e | Frequencia |
|------|---------|-----------|
| Refresh | Moderniza execucao, mantem essencia | 5-10 anos |
| Rebrand | Reformulacao significativa | Quando necessario |
| Renaming | Mudanca de nome | Ultimo recurso |

## FRAMEWORKS DE ELITE

### Brand Wheel (Unilever)
```
VALORES & PERSONALIDADE -> BENEFICIOS EMOCIONAIS -> DISCRIMINADOR -> ESSENCIA
```

### Kapferer Brand Identity Prism
```
PHYSIQUE | PERSONALITY | RELATIONSHIP | CULTURE | REFLECTION | SELF-IMAGE
```

### Brand Resonance Pyramid (Keller)
```
4. RESONANCE (fidelidade, comunidade)
3. JUDGMENTS + FEELINGS
2. PERFORMANCE + IMAGERY
1. SALIENCE (awareness)
```

### Brand Equity (Keller/Aaker)
1. Brand Awareness — As pessoas conhecem-te?
2. Brand Associations — O que associam a tua marca?
3. Perceived Quality — Acreditam que es bom?
4. Brand Loyalty — Voltam e recomendam?

## EXERCICIOS DE ELITE

1. **Brand Obituary** — Imagina que a empresa fecha amanha. Escreve o obituario. O que se perde?
2. **Brand Box** — Se a marca fosse uma caixa fisica, como seria? Tamanho, peso, material, preco, onde na prateleira?
3. **Dinner Party Test** — A marca e convidada para um jantar. Como se comporta? Com quem se senta?
4. **Iceberg Model** — O que o cliente ve (10%) vs. o que nao ve (90%).
5. **3 Words** — 3 palavras que descrevem. 3 que NUNCA poderiam ser.

## ERROS FATAIS

### Estrategia
- Tentar ser tudo para todos
- Copiar os lideres do setor
- Estrategia como exercicio academico, nao de negocio
- Saltar direto para o visual

### Identidade Visual
- Seguir tendencias de design (envelhecem)
- Paletas com demasiadas cores (max 4-5)
- Demasiados pesos tipograficos (max 2 fontes)
- Logotipo complexo demais (deve funcionar em 1x1cm a P&B)

### Implementacao
- Inconsistencia ao longo do tempo e touchpoints
- Dizer sem fazer (valores vs comportamentos)
- Lancar sem alinhamento interno
- Nao ter guardiao da marca

## NUMEROS QUE IMPORTAM

- Marcas fortes comandam premium de preco de 20-30%
- Consumidores sao 4x mais propensos a comprar de marcas com proposito forte
- Marca consistente aumenta receita em 10-20%
- Branding forte = 50% menos custo de aquisicao de clientes
