# Copy Framework Selection — `/landing copy <type>`

> **Agent:** Teresa (Sales Copywriter) | **Framework:** Schwartz 5 Levels + 8 Copy Frameworks

## Framework Selection by Awareness Level

| Awareness | Framework | Structure |
|-----------|-----------|-----------|
| Unaware | Star-Story-Solution | Protagonist → Journey → Resolution |
| Problem-Aware | PAS | Problem → Agitate → Solve |
| Solution-Aware | PASTOR | Problem → Amplify → Story → Transformation → Offer → Response |
| Product-Aware | AIDA | Attention → Interest → Desire → Action |
| Most Aware | 4 U's | Urgent, Unique, Ultra-specific, Useful |

## The 8 Frameworks

**AIDA:** Attention → Interest → Desire → Action

**PAS:** Problem → Agitate → Solve

**PASTOR:** Problem → Amplify → Story → Transformation → Offer → Response

**PPPP:** Promise → Picture → Proof → Push

**BAB:** Before → After → Bridge

**StoryBrand SB7:** Character → Problem → Guide → Plan → CTA → Failure → Success

**Star-Story-Solution:** Introduce star → Tell their story → Reveal the solution

**4 U's:** Make it Urgent, Unique, Ultra-specific, Useful

## Headline Rules (David Ogilvy)
- Headlines = 80% of the work
- Include the key benefit
- Use numbers when possible
- Ask a question the reader wants answered
- Write 10+ headlines, pick the best

## Human Writing Standard (NON-NEGOTIABLE)
- No AI patterns: no "leverage", "utilize", "robust", "streamline"
- No em-dashes as sentence connectors
- Active voice by default
- Concrete facts over vague qualifiers
- Varied sentence length and structure
