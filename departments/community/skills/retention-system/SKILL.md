# Retention System — `/community retain`

> **Agent:** Beatriz + Maria | **Framework:** Member Lifecycle + Engagement Loops

## Member Lifecycle (6 Stages)

```
AWARE → JOINING → PARTICIPATING → CONTRIBUTING → LEADING → ADVOCATING
```

Each stage needs specific activation triggers to move members forward.

## 7 Retention Habits

1. **Daily check-in ritual** — Morning signal, daily tip, or discussion prompt
2. **Weekly event** — Live call, AMA, workshop, or challenge
3. **Monthly milestone** — Achievement recognition, level-up, badge
4. **New member welcome** — Personal greeting within 24h
5. **Win sharing** — Members share results publicly
6. **Accountability pairs** — Members paired for mutual accountability
7. **Exclusive content drip** — New value every week (never feels static)

## Churn Signals & Response

| Signal | Action | Timing |
|--------|--------|--------|
| No login 7 days | "Miss you" DM | Automated |
| No login 14 days | Personal outreach from manager | Manual |
| No login 30 days | Win-back offer or survey | Automated |
| Engagement dropping | Feature new content, invite to event | Proactive |
| Negative feedback | Direct response from lead | Within 24h |

## Gamification Elements

- Points for participation (posts, reactions, events attended)
- Levels with unlock (Level 5 = access to premium channel)
- Leaderboard (monthly top contributors)
- Badges for milestones (1st post, 10 posts, 100 posts, helper badge)
