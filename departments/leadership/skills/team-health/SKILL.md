# Team Health Check — `/lead team-build <context>`

> **Agent:** Rodrigo (Leadership Director) | **Framework:** Five Dysfunctions (Patrick Lencioni)

## The Pyramid (assess bottom to top)

### 1. TRUST (Foundation)
- Can team members admit mistakes openly?
- Do they share personal struggles?
- Is there psychological safety?
**Assessment:** Ask team "When was the last time someone admitted they were wrong?"

### 2. CONFLICT
- Does the team have real debates about ideas?
- Or is there artificial harmony (everyone agrees to avoid tension)?
**Assessment:** Observe meetings — are people sharing real opinions or nodding along?

### 3. COMMITMENT
- After discussions, does everyone buy in to decisions?
- Or do people leave meetings and ignore what was decided?
**Assessment:** Ask "What did we decide last meeting?" — if answers vary, there's no commitment.

### 4. ACCOUNTABILITY
- Do peers hold each other accountable?
- Or does only the leader confront poor performance?
**Assessment:** "When a teammate misses a deadline, who addresses it?"

### 5. RESULTS
- Is the team focused on collective results?
- Or do individuals prioritize personal status/career?
**Assessment:** "What's the team's #1 goal?" — if answers differ, results aren't shared.

## Scoring

| Dysfunction | Score 1-5 | Status |
|-------------|-----------|--------|
| Trust | | |
| Conflict | | |
| Commitment | | |
| Accountability | | |
| Results | | |

**20-25:** Healthy team. **15-19:** Work needed. **10-14:** Serious issues. **5-9:** Dysfunctional.

## Fix Order
Always fix from the bottom up. Trust first. Can't have healthy conflict without trust. Can't have commitment without healthy conflict. And so on.
