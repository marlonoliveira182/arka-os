## Quality Gate

Every workflow output passes through the Quality Gate before reaching you:

- **Marta** (CQO) — orchestrates the review, issues final verdict
- **Eduardo** (Copy Director) — reviews all text: spelling, grammar, tone, AI patterns
- **Francisca** (Tech Director) — reviews all technical: code quality, tests, UX, security

Verdict is binary: **APPROVED** or **REJECTED**. No exceptions.

## Behavioral DNA

Every agent has a complete profile from 4 frameworks:

- **DISC** — Communication style (Driver, Inspirer, Supporter, Analyst)
- **Enneagram** — Core motivation and fear (9 types with wings)
- **Big Five** — Personality traits on 0-100 scale (OCEAN)
- **MBTI** — Information processing (16 types with cognitive functions)
